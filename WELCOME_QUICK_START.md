# 🙏 स्वागत — AI Brain India

A 1-page quick start. Hand this to a solo advocate friend. They can be up and running tonight.

---

## What you're getting

A free terminal-based practice OS for Indian solo advocates. Your laptop becomes a small private legal office assistant. Matters, citations, court info, BCI Rule 36 firewall, DPDP compliance flagging — all on your own machine. No subscription. No cloud upload of your client data. MIT licensed.

## Three things to install (30 minutes total)

### 1. Python (one-time)
- **Windows:** download from https://www.python.org/downloads — IMPORTANT: check the "Add to PATH" box at install
- **Mac:** already installed (or `brew install python@3.12`)

### 2. The tool itself
Open Terminal (Mac) or PowerShell (Windows):
```
pip install git+https://github.com/Wolfgangrush/ai-law-firm-india.git
```

### 3. An AI brain (pick ONE)
- **For client work — Ollama (free, local, private):**
  - Install from https://ollama.com/download
  - Run: `ollama pull qwen3:14b`
- **For drafting/study — DeepSeek (~₹100/month):**
  - Sign up at https://platform.deepseek.com
  - **CRITICAL:** Settings → Privacy → turn OFF "Improve the model for everyone"
  - Create API key

```
ailawfirm-india
```

(Shows the welcome banner.)

Then:
```
> tell me about your High Court

> check this language: "Best advocate in Mumbai"

```

## Your data lives at

- **Mac:** `~/.ailawfirm-india/`
- **Windows:** `C:\Users\YourName\.ailawfirm-india\`

Back this up to a USB drive or cloud-sync folder. That's your entire practice in one folder.
