## How to contribute (5-minute version)

### Easiest — open an Issue

1. Go to https://github.com/Wolfgangrush/ai-law-firm-india/issues
2. Click "New Issue"
3. Title: `[translation-fix] <language> — <brief description>`
4. Body: paste the line(s) that need fixing + suggest a better version
5. Submit. We'll patch it.

### Better — open a Pull Request

1. Fork the repo on GitHub
2. Edit the file directly in GitHub (pencil icon · works in browser, no Git needed)
3. Replace the lines you want to fix
4. Add a commit message: `fix(<language>): polish <section name>`
5. Open Pull Request → we review and merge

### Best — translate a section we missed

If you want to add a section that doesn't exist in your language guide yet — go for it. Open a PR. We'll merge if the translation matches the structural meaning of the English original.

## What we need most

Native speakers' help on these specific surfaces, in priority order:

### Priority 1 — Legal terminology
The English originals use Indian legal terms like:
- "Writ Petition" · "Public Interest Litigation" · "Special Leave Petition"
- "Bar Council Rule 36 firewall" · "DPDP Act 2023 compliance"
- "Limitation Act 1963" · "Cause List" · "Order on Interim Relief"
- "Manupatra" · "SCC OnLine" · "AIR"
- "BNSS" · "BNS" · "BSA"

In each Indian language, these have:
- A standard legal-Hindi/Marathi/Bengali equivalent (e.g. Hindi: "रिट याचिका" for Writ Petition)
- An everyday colloquial form
- Sometimes just the English term used as-is (loanword)

We need a native legal practitioner to pick which form a solo advocate actually uses. Right now the AI-assisted versions sometimes pick the wrong register.

### Priority 2 — Idiom and tone
The English original has a voice: warm, peer-to-peer, "I see you, take this rope." That voice should land identically in each language. Where the AI translation feels stiff, sales-y, or textbook-y, please reword for the way solo advocates actually talk.

### Priority 3 — Honorifics
- Hindi/Urdu: "अधिवक्ता जी" / "Saab"
- Marathi: "वकील साहेब"
- Bengali: "উকিল বাবু" / "অ্যাডভোকেট সাহেব"
- Tamil: "வழக்கறிஞர்" / "Sir"
- (and so on)

We picked one form per language. If your region uses different — let us know.

### Priority 4 — Script choice
For Punjabi: we used Gurmukhi (the dominant script in Indian Punjab). For lawyers in Pakistan or other regions using Shahmukhi — we'd welcome a parallel translation if there's demand.

For Urdu: not yet included. If there's demand from solo advocates practising in Urdu (J&K · parts of UP · etc.) please open an issue and we'll add it.

## What we WON'T accept (Bar Council Rule 36 firewall)

We cannot accept translations that:
- Frame the tool as a service offering ("Hire us for...")
- Use superlatives about any specific advocate or firm ("the best in your city...")
- Solicit work for any specific practitioner
- Make claims about legal outcomes the tool produces

Rule 36 applies to translated content too. If you're translating a section that originally said "this tool helps you stay organized," don't change it to "this tool will win your cases for you." Translation = preserve meaning, not amplify.

## Why this matters

Indian solo advocates are scattered across 28 states + 8 UTs · ~22 official languages + 100+ unofficial ones. A solo advocate in Bhubaneswar who reads Odia comfortably but English with effort should get the same warm onboarding as one in Bangalore who reads English natively.

The AI-assisted translations get us 70% of the way there. Native speakers close the last 30% — the part that makes the guide feel like it was written FOR them, not AT them.

Your one PR could save another solo advocate's first 30 minutes from feeling like a homework assignment.
