## ✨ ਇਹ ਕੀ ਹੈ, ਇੱਕ paragraph ਵਿੱਚ

ਇੱਕ ਮੁਫ਼ਤ tool ਜੋ ਤੁਹਾਡੇ laptop ਨੂੰ ਇੱਕ ਛੋਟਾ ਨਿੱਜੀ ਕਾਨੂੰਨੀ office assistant ਬਣਾਉਂਦਾ ਹੈ। ਇਹ ਤੁਹਾਡੇ ਸਾਰੇ matters ਯਾਦ ਰੱਖਦਾ ਹੈ (ਹਰ case file, parties, hearings, orders, dates)। ਇਹ ਭਾਰਤੀ legal citations parse ਕਰਦਾ ਹੈ (AIR · SCC · SCC OnLine)। ਕਿਹੜੀ court ਦਾ ਕਿੱਥੇ jurisdiction ਹੈ ਦੱਸਦਾ ਹੈ। BCI Rule 36 risks flag ਕਰਦਾ ਹੈ। ਇਹ **ਤੁਹਾਡੇ ਆਪਣੇ computer ਉੱਤੇ** ਚੱਲਦਾ ਹੈ — cloud ਨਹੀਂ, upload ਨਹੀਂ, ਤੀਜੀ ਧਿਰ ਤੁਹਾਡੇ client data ਨਹੀਂ ਪੜ੍ਹ ਸਕਦੀ। Open-source (MIT license)।

## 💛 ਭਾਰਤ ਦੇ solo ਵਕੀਲਾਂ ਲਈ ਕਿਉਂ

- **ਘੱਟ ਭੁੱਲੋਗੇ** — ਮਹੀਨਿਆਂ ਪੁਰਾਣੇ matter ਦਾ context ਤੁਰੰਤ ਵਾਪਸ ਆਉਂਦਾ ਹੈ।
- **ਘੱਟ deadlines miss ਹੋਣਗੀਆਂ** — hearing dates ਅਤੇ limitation periods track ਹੁੰਦੀਆਂ ਹਨ।
- **BCI Rule 36 safe** — publishing ਤੋਂ ਪਹਿਲਾਂ firewall ਤੁਹਾਡੀ language ਦੀ ਜਾਂਚ ਕਰਦਾ ਹੈ।
- **ਪੈਸਾ ਬਚੇਗਾ** — Manupatra ₹15,000+/ਸਾਲ। ਇਹ ₹0।
- **Private ਰਹੋਗੇ** — client matters laptop ਤੋਂ ਬਾਹਰ ਨਹੀਂ ਜਾਂਦੇ।

## 🚀 30 ਮਿੰਟਾਂ ਵਿੱਚ ਸ਼ੁਰੂਆਤ

### Step 1 — Python install
- **Mac:** [MAC_INSTALL.md](MAC_INSTALL.md) ਦੇਖੋ
- **Windows:** [WINDOWS_INSTALL.md](WINDOWS_INSTALL.md) ਦੇਖੋ

### Step 2 — Tool install

Terminal (Mac) ਜਾਂ PowerShell (Windows) ਖੋਲ੍ਹ ਕੇ paste ਕਰੋ:

```
pip install git+https://github.com/Wolfgangrush/ai-law-firm-india.git
```

### Step 3 — AI brain ਜੋੜੋ

| Choice | Cost | Privacy | ਕਿਸ ਲਈ |
|---|---|---|---|
| 🥇 **Local (Ollama + Qwen3)** | ₹0 ਹਮੇਸ਼ਾ ਲਈ | 🟢 ਸੰਪੂਰਨ | **client matters handle ਕਰਨ ਵਾਲੇ ਸਾਰਿਆਂ ਲਈ** |
| 🥈 **DeepSeek API** | ~₹80-200/ਮਹੀਨਾ | ⚠️ ਚੰਗੀ ਪਰ opt-out ਚਾਹੀਦਾ | client ਨਹੀਂ ਕੰਮ |
| 🥉 **Claude / Gemini API** | ~₹1500-5000/ਮਹੀਨਾ | 🟢 ਮਜ਼ਬੂਤ | heavy daily users |

[MODEL_SETUP.md](MODEL_SETUP.md) ਦੇਖੋ।

### Step 4 — ਪਹਿਲੀ ਗੱਲਬਾਤ

```
ailawfirm-india
```

ਇਹ try ਕਰੋ:
```
> tell me about Punjab and Haryana High Court
> validate AIR 1973 SC 1461
> check this language: "Best advocate in Chandigarh"
> what's the limitation period for a money suit?
```

## 🛡️ Privacy ਦੀ ਇਮਾਨਦਾਰ ਗੱਲ

**Tool ਆਪ:** ਸਾਰਾ data ਤੁਹਾਡੇ laptop ਉੱਤੇ `~/.ailawfirm-india/` folder ਵਿੱਚ। ਕਿਸੇ cloud ਉੱਤੇ upload ਨਹੀਂ।

**Cloud AI:** Cloud API ਵਰਤੋ ਤਾਂ queries ਉਹਨਾਂ ਦੇ servers ਉੱਤੇ ਜਾਂਦੀਆਂ ਹਨ। ਅਸਲੀ client ਗੁਪਤਤਾ ਲਈ **local Ollama** ਵਰਤੋ।

## 📁 Data ਕਿੱਥੇ ਰਹਿੰਦਾ ਹੈ

```
~/.ailawfirm-india/                  ← Mac
C:\Users\YourName\.ailawfirm-india\  ← Windows
├── palace/                          ← ਸਾਰੀ memory
├── config.json                      ← settings
└── people_map.json                  ← optional aliases
```

Backup: ਇਸ folder ਨੂੰ USB / Dropbox / OneDrive ਉੱਤੇ copy ਕਰੋ।
