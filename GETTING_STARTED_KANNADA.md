## ✨ ಇದು ಏನು, ಒಂದು paragraph ನಲ್ಲಿ

ಒಂದು ಉಚಿತ tool — ನಿಮ್ಮ laptop ಅನ್ನು ಒಂದು ಸಣ್ಣ ಖಾಸಗಿ ಕಾನೂನು office assistant ಆಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ. ಇದು ನಿಮ್ಮ ಎಲ್ಲಾ matters ಅನ್ನು ನೆನಪಿಡುತ್ತದೆ (ಪ್ರತಿ case file, parties, hearings, orders, dates). ಇದು ಭಾರತೀಯ legal citations ಅನ್ನು parse ಮಾಡುತ್ತದೆ (AIR · SCC · SCC OnLine). ಯಾವ court ಗೆ ಯಾವ jurisdiction ಎಂದು ಹೇಳುತ್ತದೆ. BCI Rule 36 risks ಅನ್ನು flag ಮಾಡುತ್ತದೆ. ಇದು **ನಿಮ್ಮ ಸ್ವಂತ computer ನಲ್ಲಿ** ರನ್ ಆಗುತ್ತದೆ — cloud ಇಲ್ಲ, upload ಇಲ್ಲ, ಮೂರನೇ ಪಕ್ಷ ನಿಮ್ಮ client data ಓದಲು ಸಾಧ್ಯವಿಲ್ಲ. Open-source (MIT license).

## 💛 ಭಾರತದ solo ವಕೀಲರಿಗೆ ಏಕೆ

- **ಕಡಿಮೆ ಮರೆಯುತ್ತೀರಿ** — ತಿಂಗಳುಗಳ ಹಳೆಯ matter context ತಕ್ಷಣ ಮರಳಿ ಬರುತ್ತದೆ.
- **ಕಡಿಮೆ deadlines miss ಆಗುತ್ತವೆ** — hearing dates ಮತ್ತು limitation periods track ಆಗುತ್ತವೆ.
- **BCI Rule 36 safe** — publishing ಗೆ ಮೊದಲು firewall ನಿಮ್ಮ language ಪರಿಶೀಲಿಸುತ್ತದೆ.
- **ಹಣ ಉಳಿಯುತ್ತದೆ** — Manupatra ₹15,000+/ವರ್ಷ. ಇದು ₹0.
- **Private ಆಗಿರುತ್ತೀರಿ** — client matters laptop ಬಿಟ್ಟು ಹೊರಗೆ ಹೋಗುವುದಿಲ್ಲ.

## 🧠 ಒಳಗೆ 7 specialists ಇರುತ್ತಾರೆ

| # | Specialist | ಏನು ಮಾಡುತ್ತಾರೆ |

### Step 1 — Python install
- **Mac:** [MAC_INSTALL.md](MAC_INSTALL.md) ನೋಡಿ
- **Windows:** [WINDOWS_INSTALL.md](WINDOWS_INSTALL.md) ನೋಡಿ

### Step 2 — Tool install

Terminal (Mac) ಅಥವಾ PowerShell (Windows) ತೆರೆದು paste ಮಾಡಿ:

```
pip install git+https://github.com/Wolfgangrush/ai-law-firm-india.git
```

| 🥈 **DeepSeek API** | ~₹80-200/ತಿಂಗಳು | ⚠️ ಒಳ್ಳೆಯದು ಆದರೆ opt-out ಬೇಕು | client ಅಲ್ಲದ ಕೆಲಸ |
| 🥉 **Claude / Gemini API** | ~₹1500-5000/ತಿಂಗಳು | 🟢 ಬಲವಾದ | heavy daily users |

### Step 4 — ಮೊದಲ ಸಂಭಾಷಣೆ

```
ailawfirm-india
```

ಇವುಗಳನ್ನು try ಮಾಡಿ:
```
> tell me about Karnataka High Court
> validate AIR 1973 SC 1461

> what's the limitation period for a money suit?
```

## 🛡️ Privacy ಯ ಪ್ರಾಮಾಣಿಕ ಪಕ್ಷ

**Tool ಸ್ವತಃ:** ಎಲ್ಲಾ data ನಿಮ್ಮ laptop ನಲ್ಲಿ `~/.ailawfirm-india/` folder ನಲ್ಲಿ. ಯಾವುದೇ cloud ಗೆ upload ಇಲ್ಲ.

**Cloud AI:** Cloud API ಬಳಸಿದರೆ queries ಅವರ servers ಗೆ ಹೋಗುತ್ತವೆ. ನಿಜವಾದ client ಗೌಪ್ಯತೆಗಾಗಿ **local Ollama** ಬಳಸಿ.

## 📁 Data ಎಲ್ಲಿರುತ್ತದೆ

```
~/.ailawfirm-india/                  ← Mac
C:\Users\YourName\.ailawfirm-india\  ← Windows
├── palace/                          ← ಎಲ್ಲಾ memory
├── config.json                      ← settings
└── people_map.json                  ← optional aliases
```

Backup: ಈ folder ಅನ್ನು USB / Dropbox / OneDrive ಗೆ copy ಮಾಡಿ.
